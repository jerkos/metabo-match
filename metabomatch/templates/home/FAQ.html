F.A.Q

Send us an email if you have any questions.

Can I add the software I wrote on metabomatch ?
Of course. Just go to the register software page. Fill a maximum of fields and save it. Feel free to comment and upvote it !

How software gloabl rates are computed ?
We compute a software rate using several informations about it:

  • useres mean rate
  • users upvotes
  • github statistics (#commits, #maintainers, #issues...)
  • software has been publicated or not
  • software is still maintained
  • Source code available or not

Why can we upvote a software several times a day ? Is it not cheated ?
Indeed, original idea was to allow user to upvote only like 5 times a day. For the moment, we preferred give users POWER ! Upvotes count will be reset at the end of each month.

What are the advantages if I create an account on metabomatch ?
You could post questions on our forums (be notified when a discussion has a new comment), post a comment (with a fun objectivity analysis) and/or a rate on a software, read all users scripts about optimal software parameters, little programs, and finally read/post job offers.
