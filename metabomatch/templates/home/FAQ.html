F.A.Q

Send us an email if you have any questions.

What is the purpose of metabomatch ?
The purpose of metabomatch is to help users to choose the most appropriate tool for their needs. More details here.


Can I add the software I wrote on metabomatch ?
Of course. Just go to the register software page. Fill a maximum of fields and save it. Feel free to comment and upvote it !


My software does not fit in any category. What do I do ?
Just send us a mail. We will do the necessary.


How software gloabl rates are computed ?
We compute a software rate using several informations about it:

  • users mean rate
  • users upvotes
  • # users using a software
  • Source code available or not
  • Github statistics (#commits, #maintainers, #issues...). Plan to support other version control like Sourceforge and Bitbucket.
  • software is still maintained
  • software has been publicated in a scientific journal or not
